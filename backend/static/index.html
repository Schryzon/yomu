How it Works

1

Install

Add the yōmu! extension to your browser or use the web demo.

2

Scan

Navigate to any site. yōmu! automatically detects supported languages.

3

Read

Click the widget to see pronunciation guides and deep grammar analysis.

On the Go?

You can use yōmu! on your mobile devices too:

  • iOS: Install our Safari Extension to get furigana on any webpage.
  • Arabic, Russian & Hindi: Get instant Latin transliteration for Cyrillic, Arabic, and Devanagari scripts.
  • Web Demo: Save this page to your home screen for quick access.
